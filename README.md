# Do G7 Climate Commitments Match Reality
From Global Pledges to Measurable Action (2015–2024)

## A Climate Policy Analytics Project

## Executive Summary

Climate change remains one of the most urgent global challenges, and the G7 nations continue to position themselves as leaders in climate governance. Through annual summits and international agreements, these countries have pledged ambitious emissions reduction targets for 2030.
This project evaluates whether those commitments align with actual emissions trends.

Using emissions data from 2015 to 2024, this analysis compares each G7 country's progress against its 2030 climate targets to measure policy effectiveness, accountability, and implementation gaps.

The analysis combines : Google Sheets, Power BI, and Python to build a multi-layered policy intelligence framework.

## Core Finding
Despite strong public commitments, no G7 country is currently on track to meet its 2030 target.
